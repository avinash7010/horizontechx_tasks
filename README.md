# E-Commerce Store

Functional Django e-commerce store with product catalog, cart, authentication, checkout, and order tracking.

## Run locally

1. Create and activate a virtual environment.
2. Install dependencies with `pip install -r requirements.txt`.
3. Run database migrations with `python manage.py migrate`.
4. Start the app with `python manage.py runserver`.

## Features

- Product listing and product detail pages
- Shopping cart stored in session
- User registration and login
- Order placement and tracking
- SQLite-backed users, products, and orders
